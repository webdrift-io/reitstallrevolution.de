- **HTML5/CSS3** (Tailwind CSS for styling)
- **JavaScript** (Lucide icons, GSAP for animations)
- **Static Hosting** (Compatible with GitHub Pages)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser to view the site.
3. No build step is required for the static version.

## Hosting
This site is ready for hosting on **GitHub Pages**. Simply enable Pages in the repository settings and point it to the `main` branch.
